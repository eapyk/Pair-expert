# Pair-expert

## ❓ Question

If you have a sorted list of **1024 elements**, what is the **maximum number of steps** needed in **binary search**?

---

## 🧠 Solution

Binary search reduces the search space by half every step.

We calculate:

log₂(1024)

Since:
2¹⁰ = 1024

---

## ✏️ Step-by-step

1024 → 512 → 256 → 128 → 64 → 32 → 16 → 8 → 4 → 2 → 1

That is **10 steps**

---

## 🚀 Final Answer

✔️ **10 steps**
